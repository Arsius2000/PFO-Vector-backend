## Эндпоинты

### Пользователь
| Метод | Путь | Описание |
|-------|------|----------|
| `POST` | `/users/add` | Добавление пользователя |
| `POST` | `/users/import-users` | Импорт excel таблицы с пользователями |
|`GET` | `/users/{user_id}` | Получение пользователся по user_id |
| `GET` | `/users/all` | Получение всех пользователей |
| `GET` | `/users/profile/{telegram_username}` | Вывод пользователя по telegram_username |
| `GET` | `/users/all/{role}` | Вывод списка пользователей по ролям |
| `PATCH` | `/users/profile/{user_id}` | Обновление данных пользователя |
| `DELETE` | `/users/{user_id}` | Удаление пользователя по id |

### Мероприятия
| Метод | Путь | Описание |
|-------|------|----------|
| `POST` | `/events/add` | Добавление мероприятия |
| `GET` | `/events/all` | Получение всех мероприятий |
| `GET` | `/events/{event_id}` | Получение мероприятия по id |
| `GET` | `/events/split/all` | Получение мероприятий разделённых на предстоящие/прошедшие |
| `GET` | `/events/split/{status}` | Получение мероприятий по их статусу (предстоящие/прошедшие) |
